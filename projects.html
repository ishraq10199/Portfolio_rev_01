Typing Tutor

A terminal based C program for improving typing speeds.
Details:
Made as a CSE 1st year project, Typing Tutor is a command line program that can take a text file full of words and create a typing game out of it. Typing speed increase is the purpose of this game. [Github Repo]

FilePhile

A simple web app for file management.
Details:
Made as a CSE 2nd year project, FilePhile is a naive way to manage files. It uses Firebase for storing file paths, but stores the data in a local storage, without encryption. [Github Repo]

Project Medica

A webapp that aims to automate the appointment creation and prescription system.
Details:
Made as a CSE 3rd year project, Project Medica uses raw PHP and Oracle XE 11g database to provide an automated login-based system for patients, doctors and caregivers to access medical services more easily. [Github Repo]
